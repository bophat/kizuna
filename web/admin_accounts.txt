# THÔNG TIN TÀI KHOẢN QUẢN TRỊ (ADMIN ACCOUNTS)

---

## 1. Website & Admin Dashboard
Cả ứng dụng frontend (website) và trang admin dashboard đều dùng chung tài khoản quản trị (được cấp từ backend Django).
Tài khoản này có toàn quyền trong hệ thống.

- **Email**: `admin@example.com`
- **Username**: `admin`
- **Password**: `admin123`  *(Đã được đặt lại về mật khẩu này để bạn dễ dàng sử dụng)*

Đường dẫn đăng nhập (khi project đang chạy qua Docker):
- Trang Admin Dashboard (Frontend): http://localhost:3001
- Website (Frontend): http://localhost:3000
- Trang Backend Admin mặc định (Django): http://localhost:8000/admin

## 2. Database (Cơ sở dữ liệu)
Dự án đang sử dụng **SQLite** (một cơ sở dữ liệu dạng file cục bộ), do đó **không yêu cầu cấu hình tài khoản (username/password)** để truy cập. 

Bạn có quyền quản trị toàn bộ dữ liệu chỉ bằng cách mở file sau với các công cụ đọc DB:
- **Đường dẫn DB**: `database/db.sqlite3`
- **Cách mở**: 
  - Mở trực tiếp bằng Extension "SQLite Viewer" trên VS Code.
  - Sử dụng các phần mềm chuyên quản trị như "DB Browser for SQLite", "DBeaver", "DataGrip".
